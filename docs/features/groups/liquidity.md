# 流动性

## 业务含义

这组特征描述成交量、成交额、换手与量能冲击，包括原生流动性输入和在其上做出的滚动压缩特征。

## 建模角色

- 同时包含原子输入与 aggregated 特征
- 原生量价字段更适合 sequence 主输入
- 放量冲击、滚动 zscore 更偏 tabular

## 典型用途

- 判断短线资金参与程度
- 区分容量与冲击
- 作为树模型的压缩流动性画像

## 与其他组的边界

- 不负责价格结构，那属于 `price` 或 `microstructure`
- 不负责横截面相对强弱，那属于 `cross_section`

## 使用注意事项

- 流动性组内部不要混淆原生字段和滚动压缩列的角色
- sequence 主通道应优先保留原生流动性输入
