# 价格

## 业务含义

这组特征描述最基础的价格状态，包括原生 OHLC、VWAP、复权因子，以及直接依赖价格路径的正式输入列。

## 建模角色

- 主要角色：原子输入
- 时间属性：动态日频
- 对 sequence 的意义：最高
- 对 tabular 的意义：高

## 典型用途

- sequence / Transformer 主输入通道
- tabular 模型的基础价格底座
- 所有派生价格特征的上游依赖

## 与其他组的边界

- 不负责成交量、成交额和换手率，那属于 `liquidity`
- 不负责 K 线形态压缩，那属于 `microstructure`
- 不负责可交易性约束，那属于 `tradability`

## 使用注意事项

- 这是未来多模型共享的 source of truth 主路径
- 不应因为研究方便而把它和大量压缩派生列混成一层
