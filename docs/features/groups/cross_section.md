# 横截面

## 业务含义

这组特征关注“相对别人强不强”，包括：

- 近几日收益与均量/均额
- 横截面 rank
- 相对指数超额收益
- 相对行业超额收益
- 行业内位置
- Alpha158 中更偏趋势、排序和比较的表达

## 建模角色

- 主要是 aggregated / relative / comparison 特征
- 更偏 tabular 主排序层

## 典型用途

- 做横截面主排序
- 研究相对强弱来源
- 结合行业背景降低风格误判

## 使用注意事项

- 这组很适合树模型，但不应取代原子价格输入
- 新增行业/指数相对特征应继续放在这组，而不是再拆回历史命名
